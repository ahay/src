1. Creating a branch

 svn copy  https://svn.code.sf.net/p/rsf/code/trunk  https://svn.code.sf.net/p/rsf/code/branches/madagascar-1.7 -m 'new release'

2. Check out the branch

svn co https://svn.code.sf.net/p/rsf/code/branches/madagascar-1.7

3. cd madagascar-1.7

4. Edit VERSION.txt to remove -svn

5. Remove junk

svn delete book/Grave and user/nobody

6. Make sure the book directory structure is consistent 

cd book
ls */*/*/*/SConstruct

should not return anything

7. Remove book directories without reproducible examples

import os, glob

for dir in glob.glob('*/*/[a-z]*/'):
    if os.path.isdir(dir):
       proj = os.path.join(dir,'.rsfproj')
       if not os.path.isfile(proj):
          print 'no project in ',dir

8. Compile

cd madagascar-1.7
export RSFROOT=$HOME/mad17

remove instances of previous RSFROOT from PATH and PYTHONPATH

./configure
make install

9. Remove programs without examples 

python admin/cleanup.py > list.txt

edit list.txt

svn remove `cat list.txt`

10. Clean individual directories

cd system/generic

python ../../admin/nomain.py > list.txt
svn remove `cat list.txt`

import glob
for main in glob.glob('M*.c'):
    print main[1:-2]

edit SConstruct

python ../../admin/doth.py > list.txt
svn remove `cat list.txt`

scons

11. Compile again from the top directory

12. Edit $PYTHONPATH/rsf/prog.py

replace

try:
    import rsf.use
except:
    pass

#try:
import rsf.use
#except:
#    pass

Try running sfdoc

13. Announce at RSF-devel

14. Test compilation

15. Create a branch of the core

svn copy  https://svn.code.sf.net/p/rsf/code/branches/madagascar-1.7 https://svn.code.sf.net/p/rsf/code/branches/madagascar-core-1.7 -m 'new release'

svn co https://svn.code.sf.net/p/rsf/code/branches/madagascar-core-1.7

cd madagascar-core-1.7

16. Clean core

svn delete system/seismic trip

svn delete book/bei book/geo384w (and everything else seismic-related
in book)

python admin/cleanup.py > list.txt

17. Make framework

cd madagascar-1.7/build/framework

cp ../../framework/setup.py .

python setup.py sdist

cp dist/dist/madagascar-framework-1.7-1.tar.gz ~

18. tar and upload to sourceforge.net

19. Announce on the blog

20. Edit $RSFSRC/VERSION.txt to 1.8-svn

21. Edit $RSFSRC/NEWS.txt to record the time of the release. Add
release notes to http://www.ahay.org/wiki/Release_Notes
